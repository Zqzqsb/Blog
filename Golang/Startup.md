## 介绍

**golang的特点**

| 特点         | 描述                   |
| ---------- | -------------------- |
| 高性能，高并发    | 接近C++的性能，标准库原生高并发支持。 |
| 语法简单，学习曲线缓 | 学习周期短至周计             |
| 丰富的标准库     | 可以解决大部分需求            |
| 工具链        | 编译，代码格式化，代码检查，测试     |
| 静态编译       | 体积小，部署编译             |
| 快速编译       | 支持增量编译               |
| 快平台        | 交叉编译简单               |
| GC         | 原生GC支持               |
